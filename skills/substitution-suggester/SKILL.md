# Instructions

If the user mentions they are missing a standard pantry item or want to swap an ingredient from the generated options:
1. Suggest 1-2 simple, common alternatives (e.g., swapping butter for olive oil, or lemon juice for vinegar).
2. Briefly explain how the substitution might slightly alter the taste or texture, but reassure the user it will still work.
3. Apply the substitution to the final recipe execution.
