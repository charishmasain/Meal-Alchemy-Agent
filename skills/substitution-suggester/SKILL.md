# Substitution Suggester

Trigger this skill when:
- The user says they don't have a specific ingredient, OR
- An ingredient in the recipe seems hard to find.

For each missing ingredient:

1. Suggest **1–2 simple, common alternatives** that are likely already in a typical home kitchen.
2. Briefly explain how the substitution may slightly change the taste or texture — and reassure the user the dish will still work.
3. Automatically apply the substitution to the recipe steps if needed.

**Example:**
> You don't have butter? No problem!
> - Use **olive oil** instead — the dish will be slightly less creamy but just as tasty.
> - Or try **coconut oil** for a mild sweetness.
