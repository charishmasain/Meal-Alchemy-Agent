# Time Optimizer

When processing any recipe:

1. Break down the total time into:
   - **Prep time** (chopping, measuring, marinating)
   - **Cook time** (active heat time)
   - **Total time** (combined)

2. Identify any steps that can be done **in parallel** to save time.
   - Example: *"While the pasta boils, chop the vegetables."*
   - Example: *"While the pan heats up, beat the eggs."*

3. Add 1–2 time-saving tips specific to the recipe being cooked.

4. Always ensure the estimated time is realistic for a beginner cook — do not underestimate.
